# 🎓 Student Management System (SQL Project)

## 📌 Overview
This project is a fully structured SQL database system designed to manage academic data, including students, departments, courses, enrollments, and instructors. It simulates a real-world education system with efficient data handling and analysis.

## 🚀 Features
- Relational database design with multiple tables
- Implementation of Primary Keys and Foreign Keys
- CRUD operations (Insert, Update, Delete, Select)
- Advanced JOIN operations (INNER JOIN, LEFT JOIN)
- Subqueries and nested queries
- Aggregate functions (COUNT, AVG)
- Window functions (Running Total)
- Conditional logic using CASE statements

## 🛠️ Technologies Used
- SQL (MySQL)
- DBMS (Database Management System)

## 📂 Database Structure
- **Departments** – Department details
- **Students** – Student information
- **Courses** – Course details
- **Enrollments** – Student-course mapping
- **Instructors** – Faculty information

## 📊 Key Operations Performed
- Managing student enrollments across courses
- Analyzing student distribution by department
- Identifying course popularity using COUNT
- Calculating average course credits
- Ranking and filtering academic data
- Performing real-world academic queries

## 🎯 Purpose
The objective of this project is to understand database relationships and apply SQL concepts in a real-world academic management system.

## 📌 Conclusion
This project demonstrates strong SQL skills including database design, data manipulation, and advanced querying, making it suitable for real-world applications like school or college management systems.
